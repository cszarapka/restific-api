# restific-api

A simple Rails API REST service that allows you to register a user and generate unique identifiers.

## Getting Started

### Prereqs

* ruby 2.6.5
* rails 5.0.7.2
* postgres (PostgreSQL) 12.2

### Install

Clone the repo and install the gems.

Create and migrate database (postgresql):

```
$ rails db:create
$ rails db:migrate
```

### Run app

`rails s`

### Run tests

`bundle exec rspec`

## Try it Out
